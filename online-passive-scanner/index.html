Online passive scanner

Quick passive reconnaissance for web targets

This scanner reviews a target URL and its DNS posture for externally visible risk indicators. It is passive-only and does not run exploit payloads against the target.

Scanner source

URL posture checks

Flags insecure schemes, exposed credentials, risky parameters, and suspicious route patterns.

DNS posture checks

Queries A, AAAA, CAA, MX, NS, and TXT records through DNS-over-HTTPS for baseline exposure signals.

Shareable report URLs

Each report can be reopened by sharing a link with the encoded `#q=` target value.
